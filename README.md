# DXCVendingMachine

A small vending machine app written in Angular.
Accepted coin denominations: Euro (1, 2, 5, 10, 20, 50 cents, €1, €2).

## Build

Install dependencies:
    ```bash
        npm install
    ```

To start a local server:
    ```bash
        ng serve
    ```

Navigate to the app at `http://localhost:4200/`
